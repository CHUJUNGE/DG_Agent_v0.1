### 模块 M4：6 天实时日记（核心模块）
- 模块目的：捕捉真实发生的事件（使用/未使用口香糖），并按链路复现决策过程与体验结果（符合日记链路：trigger → context → action → alternatives → choice logic → target category role → experience → before/during/after → evaluation）。
- 对应研究问题：RQ1–RQ6（核心产出）
- 关键观察点：触发、场景/人物/地点、替代方案、是否想到口香糖、是否购买/使用、使用体验、情绪/注意力的变化、是否产生再购买意愿
- 结构说明：建议每日 1 次“早/中/晚”触发提醒 + 事件上报（每次事件 6–8 问）；若当天无相关事件，则做一次“空记录”并询问原因（没场景/忘带/不方便）。
- 建议题量/天：平均 6–10 条（事件），每条 6–8 个小问；6 天总量控制在可接受范围内（见方案末日负荷建议）。

日记模板（每次事件/情境，受访者按此流程填报）：

1) 事件触发（trigger）：刚刚发生了什么使你产生“想让嘴巴/情绪/精神好一点”的念头？请尽量用自己的话描述当时情况（例如：刚吃完外卖；下午工作卡住；开车很疲惫）。  
   - 研究目的：确认触发类型。

2) 场景细节（context）：时间、地点、当时在做什么、有无他人在场、是否需要马上见人/上台/继续工作？  
   - 追问：当时是独自一人还是与朋友/同事？是否能随意吃东西？  
   - 素材要求：鼓励上传一张情境照（手机拍摄，若不方便可跳过）；上传后研究员可判断环境便利性。

3) 行动（action）：当时你做了什么？（选项示例：嚼口香糖 / 含薄荷糖 / 喝咖啡 / 口喷 / 什么也没做 / 去洗手间刷牙等）并简述当时选择该方案的原因。  
   - 研究目的：记录替代选择和选择逻辑。

4) 目标品类角色（target category role）：你当时是否想到口香糖？如果想到，是否马上选择？若没想到，请写出原因（例如：没带、觉得不合适、社交尴尬、现场不可得）。  
   - 追问：如果想到但没用，是哪一环阻止了你？（可多项：找不到、价格高、不想买、觉得效果不明显、怕吐掉麻烦等）

5) 体验与效果（before/during/after）：使用后你感觉怎样（立刻的效果、持续多久、有没有让你“更自在/集中/清新/镇静”）？如果没使用，请写你预期的效果是否会有帮助。  
   - 素材要求：如果使用了口香糖，请录一段 20–30 秒的语音，描述吃前和吃后感受（可选）；若可以，请拍下所用包装/品牌的照片。

6) 购买意愿与行动（evaluation）：这个体验让你是否想要马上购买/下次准备带上？（简答并说明原因）  
   - 设计说明：用于评估即时 trial-to-buy 的可能性。

（注：上面每次事件的填报控制在 3–6 分钟；若受访者愿意可扩展加录语音或短视频。）

### 模块 M5：购物任务 / 购买路径验证（一次性任务）
- 模块目的：还原购买决策、渠道可得性与收银台/货架视觉效果对购买的影响（检测小包装/价格/渠道 barrier）。
- 对应研究问题：RQ3、RQ5
- 建议题量：6–8 题（含实地拍照任务）
- 任务说明（示例）：

任务 A（线下）—— 如果你在接下来 48 小时内路过一个便利店/超市/收银台，请按以下步骤完成并上传证据：  
  1) 找到收银台/近收银处的口香糖展示并拍一张清晰照片（或如果没有，请拍店面内相近零食陈列处的照片并说明）。  
     - 研究目的：评估收银台/店内陈列的可见度与品牌优先度。  
  2) 看看有没有你平时会买的口香糖品牌（可识别品牌即可），如果你不买，请写出不买的具体原因（没有想买、没有喜欢的口味/包装、价格/小包问题、觉得不好拿/吐糖不便等）。  
     - 追问：如果当时想买，你会不会现场购买？为什么/为什么不？  
  3) （若线上）在手机上搜索“附近口香糖”或某主流外卖/即时零售平台，截屏搜索结果（包括价格/是否配送/是否方便），并说明是否会用线上方式购买。  
     - 设计说明：比较线上线下可得性差别。

任务 B（购买意向模拟）—— 给出一个小包装（单片/三片装）和一个更大包装（多片装）的图片，请你选择更可能即时购买的那一个并说明原因。  
  - 研究目的：验证小包是否能降低准入门槛。

（注：若样本无法线下访问，可调整为“回忆一次最近线下看到的情形并上传手机截图”。）

### 模块 M6：习惯养成与中断（回顾式）
- 模块目的：理解长期使用习惯（如何开始、维持、断裂）、社会传播路径和促成持续购买的关键奖励机制。
- 对应研究问题：RQ6
- 建议题量：5–7 题

示例题：

1) 回想你第一次开始经常嚼口香糖/某替代品的场景（若有），是什么动机让你开始？（例如：被朋友带动、工作需要、开车提神、口腔护理等）  
   - 研究目的：发现习惯起点的 common triggers。

2) 如果你曾经从“经常吃”变成“很少吃/不吃”，请写出造成中断的最关键原因（可列举：忘记买、觉得麻烦、社交尴尬、替代品吸引、效果不明显、健康顾虑等）。  
   - 研究目的：识别阻断机制及可设计的补救策略。

3) 对你来说，什么样的改变最可能让你重新开始并持续购买口香糖？（选项 + 开放理由：更方便的小包、更强的即时效果、更好口感/嚼感、跟某活动/社交绑定、价格/促销、好友推荐等）  
   - 设计说明：用于给 product/communication 提供优先级排序依据。

### 模块 M7：品牌与产品期待（概念方向探询，后段）
- 模块目的：在不提前暴露品牌主张的前提下，探索消费者对若干“可操作化”方向（communication 的更具体化诉求、package/format、口感/口味变体）的反应与优先级，以便产出具象方向（非终稿概念）。
- 对应研究问题：RQ2、RQ7
- 关键观察点：对“recenter”细分情绪（提神 vs 缓解紧张 vs 切换思路 vs 镇静）的可感知价值、对小包/功能提示/口感差异的接受度
- 建议题量：4–6 题；注意刺激物（若有）建议在 IDI 中深测

示例题：

1) 我们描述几类“帮助你恢复好状态”的具体场景写法（每个 1–2 句），请按你觉得“最有说服力能让你马上尝试/购买”的顺序排列，并说明原因。  
   - 场景示例（为研究员参考，不直接给受访者品牌名）：
     a) “短暂嚼一片，帮你在会议/面试前短暂镇静、缓解紧张”  
     b) “工作疲惫或脑力卡住时，嚼一片能帮你换个角度、提振注意力”  
     c) “长途驾车时帮助你保持清醒并减少分心”  
     d) “无聊时打发时间并给你一个小小的情绪修复”  
   - 研究目的：拆解“Recenter”内核的消费者优先点（哪些更具体/更可感知）。

2) 关于产品形态：你更偏好哪类（单片小包 / 多片小包装 / 盒装 / 可放口袋的薄包装 / 可随手挂链装），并说明在即时购买/随身携带时哪个最方便？  
   - 追问：如果有一款“随手放口袋的单片速效包”，你是否会更可能尝试？为什么？  
   - 设计说明：验证小包策略的可行性并收集阻力点。

3) （若允许放刺激物）给出一张新的包装草案或一句核心口号（短句），请你写下第一印象、是否信服及是否会因此更可能购买。  
   - 研究目的：初步筛选沟通语气与包装方向（若刺激物复杂，建议放 IDI）。
