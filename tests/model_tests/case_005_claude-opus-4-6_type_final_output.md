### 模块1：我是谁
模块类型：访谈题

**引导语：**
【开场白】Hi～在正式开始之前，我们想先认识一下你。没有标准答案，轻松地说说你自己就好。

**题目：**

【简答】1. 先来介绍一下你自己吧！可以用关键词、颜色、MBTI、星座，或者任何你觉得能代表自己的方式——欢迎配上照片，让我们看到真实的你。

【单选】2. 你现在是在上学、工作，还是其他状态？

【简答】3. 你的时间和精力主要花在哪些地方？哪些事情对你来说最重要，哪些是你特别享受的？

【简答】4. 你有哪些兴趣爱好？哪些是长期坚持的，哪些是最近才开始的？这些爱好通常在什么场合、和谁一起？

【简答｜必填视频】5. 带我们看看你每天待得最久的地方，或者你最用心打理的一个角落吧。拍一段小视频或几张照片都可以，真实状态就好，不用特意收拾。

【简答】6. 聊聊你的社交圈吧——你通常和哪些人来往？在不同的圈子里，你更像是发起活动的人、跟随者、组织者，还是倾听者？

【简答】7. 说到花钱，你的大头消费在哪里？有没有哪些不是必需品，但你特别愿意为它花钱的东西？为什么？

【打分】8. 如果给你现在的生活打个分（1-10分），你会打几分？扣分主要扣在哪里？

**结束语：**
【结束画面】先认识到这里。接下来聊聊你平时一天的节奏～

### 模块2：我的一天
模块类型：访谈题

**引导语：**
【开场白】接下来，聊聊你典型的一天。不用刻意美化，就是最真实的日常。

**题目：**

【简答】1. 描画一下你最近典型的一天吧——通常几点起床、几点休息？从早到晚，大概都在做什么？

【简答】2. 工作日和休息日，你的节奏有什么不同？哪些事情只在工作日发生，哪些只在休息日？

【简答】3. 你的日常里有没有一些固定的小习惯或"仪式感"时刻？比如早上的某个动作、下午的某个放松方式，或者睡前的某件事。具体说说。

【简答】4. 你一天中最有状态的时间段是什么时候？最容易分心或疲惫的时候呢？

**结束语：**
【结束画面】好的，对你的日常有感觉了。下面从书写这件事开始聊～

### 模块3：我和书写
模块类型：访谈题

**引导语：**
【开场白】这一部分想聊聊"动笔"这件事——不只是写字，画画、涂鸦、做笔记都算。

**题目：**

【简答】1. 你现在还会动笔写东西吗？在哪些情况下会写？回想一下最近一次动笔的场景——当时在哪里、在做什么、写了什么？

【简答】2. 和两三年前相比，你动笔的频率和场合有没有变化？哪些场景写得更多了，哪些写得更少了，有没有完全消失的，或者新出现的？

【简答】3. 有没有哪些事情，你觉得"一定要用笔写，不能用手机或电脑替代"？为什么？

【简答】4. 除了记录信息，动笔对你还有什么别的意义吗？比如放松、整理思路、表达自己，或者其他什么。

【简答】5. 你有没有特别享受的书写时刻？描述一下那个场景——当时的状态、在写什么、用的什么笔、感觉怎么样。

【简答】6. 反过来，有没有让你觉得"写字好麻烦"或者"不想动笔"的时候？是什么情况？

**结束语：**
【结束画面】聊完书写的感受了。接下来看看你现在都有哪些笔～

### 模块4：我的笔和文具
模块类型：访谈题

**引导语：**
【开场白】现在来认识一下你的笔吧。不管多还是少，都可以分享。

**题目：**

【简答｜必填图片】1. 拍一张你现在拥有的笔的"全家福"吧——笔袋里的、桌上的、抽屉里的，都可以。如果方便，也可以把你的文具收纳区域一起拍进来。

【简答】2. 看看这张全家福，哪支笔是你最常用的？哪支是你最喜欢的？这两个答案一样吗？为什么？

【简答】3. 有没有最近新买的笔？是什么，为什么买它？

【简答】4. 有没有买了但现在不怎么用的笔？是什么情况让它"退休"了？

【简答】5. 有没有一直想买但还没买的笔？是什么让你还没下手？

【简答】6. 聊聊你买笔的习惯吧——通常在哪里买，线上还是线下？买的时候一般怎么做决定，是冲动下单、货比三家，还是有固定偏好？大概多久买一次，每次大概花多少？

【简答】7. 你有没有特别喜欢或特别不喜欢的笔的品牌？为什么？

**结束语：**
【结束画面】你的笔的世界挺有意思的。接下来聊聊你选笔的标准～

### 模块5：我怎么选一支笔
模块类型：访谈题

**引导语：**
【开场白】每个人选笔的逻辑都不一样。这一部分想聊聊你真正在意的那些细节。

**题目：**

【简答】1. 你觉得一支"好笔"应该是什么样的？比如手感、出墨、重量、粗细、外观，或者其他你在意的地方——说说你真正看重的是什么。

【简答】2. 回想一下你用过的笔里，体验最好的一次是什么情况？当时用的是什么笔，好在哪里？尽量描述一下那种感觉——比如写起来的顺滑度、握感、声音，或者当时的心情。

【简答】3. 反过来，有没有让你特别失望或不舒服的用笔体验？是什么情况，哪里出了问题？

【简答】4. 颜值对你选笔有多重要？你理想中的笔在外观上应该是什么样的？如果有参考图片，欢迎发出来。

【简答】5. 你会在意笔的品牌吗？品牌对你来说意味着什么——品质保证、身份表达，还是其他？

【简答】6. 如果一支笔颜值很高但写起来一般，或者写感很好但外观普通，你会怎么权衡？

【简答】7. 你有没有把笔当作礼物送人，或者收到过笔作为礼物？是什么场合？你怎么看待"笔作为礼物"这件事？

**结束语：**
【结束画面】对你选笔的标准了解得差不多了。最后一个部分，聊聊你对未来的笔有什么期待～

### 模块6：我心目中的下一支笔
模块类型：访谈题

**引导语：**
【开场白】最后一个模块，也是最有想象空间的一个。来聊聊你理想中的笔。

**题目：**

【简答】1. 如果现在有一支笔，能帮你解决某个具体的书写场景或痛点，你最希望它解决的是什么？

【简答】2. 你觉得现在市面上的笔，有没有哪些地方让你觉得"一直没有做好"或者"一直没有人做"？

【简答】3. 想象一支"最能代表你"的笔——它是什么样的？外观、手感、功能、使用场景，随便说。如果有参考图片，也欢迎分享。

【简答】4. 如果有一支笔，你愿意每天随身携带、用很多年，它需要具备哪些条件？

【简答】5. 你有没有在社交媒体上看到过让你心动的笔或文具内容？是什么，为什么心动？如果有截图或链接，欢迎分享。

【简答】6. 最后，用一句话或一个词，说说"笔"对你现在的生活意味着什么。

**结束语：**
【结束画面】全部填答完成，辛苦啦！希望你今天过得愉快～
