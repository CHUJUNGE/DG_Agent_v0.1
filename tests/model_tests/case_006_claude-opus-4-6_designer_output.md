

# 题目设计方案

## 1. 项目理解

- **项目名称**：凡可奇低端猫粮消费者研究
- **品类/品牌**：低端猫粮 / 凡可奇（乐宾集团旗下，与蓝氏、茁悦同属一个品牌矩阵）
- **目标人群**：低端猫粮消费者（学生党、新晋打工人、多猫家庭、流浪猫救助等预算有限但追求"平价好粮"的养猫人）
- **商业问题**：以用户真实需求为原点，构建凡可奇品牌与产品策略——理解低端猫粮人群画像、情感驱动、猫粮需求模型、心智场景、换粮旅程与品牌切入机会
- **研究目标**：四大产出——①人群画像与情感驱动 ②猫粮需求模型与优先级矩阵 ③养宠心智场景与猫粮需求勾连 ④换粮旅程与品牌机会点
- **已有研究沉淀**：集团已完成茁悦中端猫粮消费者研究，有"人猫情感需求"分类框架和养猫契机词云等前期洞察可参考，本次不应重复泛问情感分类，而应在低端人群中验证和深化
- **关键设计依据**：Proposal 仅提取前12页，后续方法、样本、时间等信息缺失；方案基于已有信息设计，部分判断列入确认点

## 2. 核心研究问题

1. 低端猫粮消费者是什么样的人？他们的生活状态、经济压力、养猫契机和人猫关系有什么特征？
2. 他们在养猫日常中，哪些场景承载了最强烈的情感浓度（幸福/烦恼/揪心/骄傲）？这些场景如何连接到猫粮需求？
3. 他们选猫粮时真正在意什么？需求的维度、层次和优先级如何？"低端"消费者的"好粮"标准是什么？
4. 他们的换粮旅程是怎样的——从哪里获取信息、怎么比较、怎么决定、换粮的触发点和卡点是什么？
5. 当前市场上有哪些品牌/产品进入了他们的选择集？凡可奇（或同价位品牌）在其中处于什么位置？
6. 他们对未来猫粮有什么期待？什么样的品牌和产品能让他们觉得"这就是我要的平价好粮"？

## 3. 模块结构总览

| 模块 | 模块目的 | 对应研究问题 |
|------|----------|-------------|
| 模块1：关于我 | 理解受访者生活状态、经济压力、日常节奏，为后续人群画像和情感驱动提供底色 | Q1 |
| 模块2：我和我的猫 | 理解养猫契机、人猫关系、情感驱动类型，验证和深化已有情感框架在低端人群中的表现 | Q1, Q2 |
| 模块3：我的养猫日常 | 还原养猫典型一天和高情感浓度场景，连接场景与猫粮需求 | Q2 |
| 模块4：我家猫的口粮 | 下载当前猫粮全家福和使用图谱，理解选粮标准、需求维度和优先级 | Q3, Q5 |
| 模块5：我是怎么选粮换粮的 | 还原换粮旅程——信息获取、比较、决策、触发点、卡点 | Q4, Q5 |
| 模块6：我的喂食记录 | 实时日记记录1-2天喂食行为，捕捉真实场景、产品使用和情绪 | Q2, Q3 |
| 模块7：我心目中的好猫粮 | 收集未来期待、理想产品想象、品牌感知，为品牌与产品策略提供方向 | Q6 |

## 4. 详细题目设计

---

### 模块1：关于我

**引导语：**
嗨，欢迎你来到这次研究！在正式开始聊猫猫之前，我们想先认识一下你这个人。没有标准答案，怎么舒服怎么说～

**题目：**

1. 先跟我们打个招呼吧！简单介绍一下你自己——你多大了，坐标哪个城市，现在是在上学、上班还是其他状态？拍一张你觉得能代表你现在生活状态的照片，可以是你的桌面、房间一角、通勤路上，什么都行。

2. 聊聊你现在的生活节奏吧——平时一天大概是怎么安排的？忙不忙？自由时间多吗？一般会怎么度过？

3. 说到花钱这件事，你觉得自己现在的经济状况怎么样？每个月的钱大概都花在哪些地方？有没有什么是你觉得"该省省、该花花"的？

4. 生活里有没有什么事情是你最近比较操心或者有压力的？也可以聊聊最近让你开心的事。

**结束语：**
谢谢你的分享！接下来我们要聊聊你和你的猫了～

---

### 模块2：我和我的猫

**引导语：**
现在轮到你家主子登场了！我们想听听你和猫的故事。

**题目：**

5. 给我们介绍一下你的猫吧——养了几只，分别叫什么、多大了、什么品种？拍一张（或几张）你家猫的照片给我们看看。

6. 当初是怎么开始养猫的？是什么契机让你决定养猫？

7. 如果要用一个词或一句话形容你和猫之间的关系，你会怎么说？它在你生活里更像是什么角色？

8. 你觉得养猫之后，你的生活发生了什么变化？有哪些是你之前没想到的？

**结束语：**
感觉你和猫的故事很有意思！接下来我们聊聊养猫的日常。

---

### 模块3：我的养猫日常

**引导语：**
养猫的每一天都不太一样，有开心的时候，也有头疼的时候。我们想听听你真实的养猫日常。

**题目：**

9. 描述一下你平时一天里跟猫相关的事情吧——从早上起来到晚上睡觉，你大概会在什么时候跟猫互动？做些什么？

10. 养猫这件事里，有没有哪个瞬间或场景让你觉得特别幸福、特别满足？具体说说当时发生了什么。

11. 反过来，有没有哪个瞬间或场景让你特别烦恼、头疼、甚至揪心？具体说说当时发生了什么。

12. 在你养猫的过程中，有没有什么事情让你觉得特别骄傲或者有成就感的？比如猫的某个变化，或者你自己做到了什么。

13. 你觉得在养猫这件事上，你花费最多精力和心思的是哪个方面？为什么？

**结束语：**
谢谢你分享这些真实的养猫故事！接下来我们聊聊猫粮。

---

### 模块4：我家猫的口粮

**引导语：**
现在我们来聊聊你家猫吃的东西。先从你家里现在有的猫粮开始。

**题目：**

14. 把你家里现在所有的猫粮（包括主粮、零食、罐头等）都拍一张"全家福"给我们看看吧。如果有不同品牌或不同产品，都拍进来。

15. 看看你刚才拍的这些，跟我们说说：哪个是你家猫现在吃的主粮？用了多久了？

16. 你现在给猫吃的这款主粮，当初是怎么选中它的？为什么是它而不是别的？

17. 你对现在这款主粮满意吗？你觉得它哪里好、哪里不太行？

18. 除了现在吃的这款，你之前还给猫吃过哪些主粮？大概换过几次了？

19. 你选猫粮的时候，最看重的是什么？可以说说你心里的标准——什么样的猫粮你觉得是"好的"，什么样的你觉得"不行"？

20. 你一般多久买一次猫粮？每次大概花多少钱？你觉得这个花费对你来说压力大吗？

21. 你一般在哪里买猫粮？线上还是线下？具体是什么平台或店？为什么选择在那里买？

**内部提示：本模块全家福素材后续模块可引用，不需要重复拍摄。**

**结束语：**
了解了你家猫的口粮情况，接下来我们想听听你换粮的经历。

---

### 模块5：我是怎么选粮换粮的

**引导语：**
很多养猫的人都有过换粮的经历。我们想听听你的换粮故事。

**题目：**

22. 回忆一下你最近一次换猫粮的经历——是什么原因让你决定换的？当时发生了什么？

23. 你决定换粮之后，是怎么找新猫粮的？从哪里获取信息？看了些什么内容？

24. 在挑选的过程中，你有没有犹豫过、纠结过？最后是什么让你做了决定？

25. 换粮过程中有没有遇到什么困难或者不顺利的地方？比如猫不吃、买到不合适的、退换麻烦等等。

26. 你身边有没有人会跟你交流猫粮的事情？比如朋友、家人、网上的群或社区？他们的意见对你影响大吗？

27. 如果你现在想换粮，你觉得最大的顾虑是什么？什么情况下你会比较放心地去尝试一个新品牌？

**结束语：**
谢谢你分享换粮的经历！接下来有一个小任务，需要你记录一下真实的喂猫过程。

---

### 模块6：我的喂食记录

**引导语：**
接下来的1-2天里，每次给猫喂食的时候，请你记录一下当时的情况。每次喂食记录一条就好，不用攒到一起写。

**题目：**

28. 现在是什么时间？你刚才给猫喂了什么？（主粮/零食/罐头/其他）拍一张你喂食时的照片或短视频。

29. 这次喂食是在什么情况下发生的？是到了固定的饭点，还是猫主动要吃，还是你觉得该喂了，还是其他原因？

30. 喂的时候你观察到猫的反应怎么样？吃得怎么样？

31. 这次喂食过程中，你有什么感受？有没有什么让你满意或者不满意的地方？

**内部提示：本模块为日记记录模块，受访者每次喂食重复填写28-31题，建议记录1-2天、覆盖3-5次喂食。**

**结束语：**
恭喜你完成今天的填答，辛苦啦！祝你度过了美好的一天～

---

### 模块7：我心目中的好猫粮

**引导语：**
最后一个模块了！我们想听听你对猫粮的期待和想象。

**题目：**

32. 你心目中理想的猫粮是什么样的？如果不考虑价格，你最想给猫吃什么样的粮？为什么？

33. 回到现实，在你的预算范围内，你觉得一款"平价好粮"应该具备什么条件？什么是必须有的，什么是有了更好的？

34. 你有没有听说过或者关注过一些猫粮品牌？不管买没买过，说说你对它们的印象。（可以提到任何你想到的品牌）

35. 如果一个猫粮品牌想要打动你、让你愿意试试，你觉得它应该怎么做？是产品本身要怎样，还是在哪里让你看到，还是别人怎么推荐？

36. 最后，关于给猫选粮、喂猫这件事，有没有什么是你一直想说但没人问过你的？随便聊聊。

**结束语：**
所有问题都回答完了，非常感谢你的耐心和真诚！你分享的这些对我们非常有价值。祝你和你的猫都健康快乐！🐱

## 5. Wording Handoff

- **目标人群语气注意**：低端猫粮消费者可能包含学生党、刚工作的年轻人、多猫家庭、流浪猫救助者，经济压力较大但对猫有真实情感。语气应亲切、轻松、不居高临下，避免"消费升级""品质生活"等可能让受访者感到距离感的措辞。多猫家庭和救助者可能对"低端"标签敏感，题面中不要出现"低端""便宜货"等字眼。
- **不能改动/删除**：
  - 模块2中养猫契机和人猫关系题（直接服务Proposal产出1情感驱动）
  - 模块3中高情感浓度场景题（幸福/烦恼/揪心/骄傲，直接服务Proposal产出3心智场景）
  - 模块5换粮旅程完整链路（直接服务Proposal产出4）
  - 模块4中选粮标准题（直接服务Proposal产出2需求模型）
- **需要延后暴露的品牌/产品/刺激物**：凡可奇品牌名在整个DG中不出现；模块7中品牌感知题为开放式，由受访者自发提及，不主动暴露任何品牌
- **需要保留的观察点**：
  - 经济压力与花钱逻辑（理解"低端"背后的真实预算约束）
  - 人猫关系类型与情感驱动（不同于中端人群的情感特征）
  - 高情感浓度场景与猫粮需求的勾连
  - 选粮标准的维度和优先级
  - 换粮旅程中的信息获取、决策卡点和信任建立
  - "平价好粮"的具体标准
- **可能过重或过硬的题面**：
  - Q19选粮标准题可能偏抽象，wording agent可考虑拆成更具体的追问或用对比方式引导
  - Q32-33理想猫粮与平价好粮两题逻辑相近，wording agent可考虑合并或调整引导方式
  - 模块6日记记录如果要求每次喂食都记录，对多猫家庭可能负担较重，wording agent可加入"挑2-3次有代表性的记录就好"的引导
- **建议交给 dg-question-wording-editor 处理**：
  - 全卷口语化和自然度优化
  - 模块3情感场景题的引导语可以更轻松，避免像访谈提纲
  - 模块4全家福任务的拍摄引导可以更具体（比如"打开储粮的柜子拍一张"）
  - 模块6日记记录的填写频次引导需要更明确
  - 结束语统一风格调整

## 6. Agent 检核摘要

- **已对照商业问题与Proposal检核**：四大产出（人群画像+情感驱动、需求模型、心智场景、换粮旅程）均有对应模块和题目覆盖
- **主要风险**：Proposal仅提取前12页，后续方法论、样本设计、是否有IDI/入户、是否有刺激物测试等信息缺失，可能导致模块设计不完整
- **已做的控制**：凡可奇品牌名全程未暴露；先理解人再理解猫再理解猫粮的递进顺序；日记模块控制在4题/次；总题量36题，负担适中
- **已有研究沉淀处理**：茁悦项目已有情感需求分类框架，本次设计不重复泛问"你为什么养猫"的大框架，而是在低端人群中具体化情感驱动和场景
- **品牌暴露检查**：全卷无凡可奇品牌名、slogan、定位语或刺激物出现，品牌感知题为开放式自发提及

## 7. 需要确认的问题

1. **Proposal后半部分是否包含IDI/入户/焦点小组等定性方法？** 当前方案按纯Digital Diary设计；如果有IDI承接，部分深层动机题（如人猫关系深层情感、品牌人格化）可以从DG移至IDI，DG可进一步精简。

2. **是否有刺激物/产品概念/包装需要在DG中测试？** Proposal提到"助力打动人心的产品概念产出"，但未明确是否有现成概念需要受访者反馈。如果有，需要在模块7后增加刺激物测试模块。

3. **样本中是否需要区分不同养猫类型（如单猫vs多猫、自养vs救助）并设计分流题？** Proposal提到多元化宠主类型，如果需要按类型分组分析，可能需要在模块2后增加筛选/分流逻辑。
