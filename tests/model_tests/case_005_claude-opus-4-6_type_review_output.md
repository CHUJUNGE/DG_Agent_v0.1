### 模块1：关于我这个人
模块类型：访谈题

【开场白；理由：引导语为模块开头说明，不收集受访者回答。】
欢迎来到这次研究！在正式开始之前，我们想先好好认识一下你。这个模块没有标准答案，请放轻松，用你最自然的方式介绍自己就好。

【简答；理由：需要受访者用自由方式（关键词、照片等）描述自我，属于开放式画像采集。】
1. 先来介绍一下你自己吧！可以用关键词、颜色、MBTI、星座，或者任何你觉得能代表自己的方式——欢迎配上照片，让我们看到真实的你。

【简答；理由：需要受访者描述当前生活状态与日常重心，属于开放叙述。】
2. 你现在的生活状态是什么样的？上学、工作、还是其他？每天主要在忙什么？

【简答；理由：需要受访者自由描述时间精力分配与个人优先级，属于开放叙述。】
3. 你的时间和精力主要花在哪些地方？哪些事情对你来说最重要，哪些是你特别享受的？

【简答；理由：需要受访者描述兴趣爱好的具体内容、持续时间、场合与社交关系，属于多维度开放叙述。】
4. 你有哪些兴趣爱好？哪些是长期坚持的，哪些是最近才开始的？这些爱好通常在什么场合、和谁一起？

【简答｜必填视频；理由：题目要求受访者拍摄居住空间视频或照片并展示说明，属于开放叙述+必填媒体任务。设计方案明确标注此题为必填视频或必填照片，此处按视频标注。】
5. 可以带我们"云参观"一下你的居住空间吗？拍一段小视频或几张照片，展示你每天待得最久的地方，或者你最用心打理的一个角落。

【简答；理由：需要受访者描述社交圈构成与自身社交角色，属于开放叙述。】
6. 聊聊你的社交圈吧——你通常和哪些人来往？在不同的圈子里，你通常是什么角色？比如发起活动的人、跟随者、组织者，还是倾听者？

【简答；理由：需要受访者描述消费结构与非必需品消费动机，涉及具体原因和故事，属于开放叙述。】
7. 说到花钱，你的大头消费在哪里？有没有哪些不是必需品，但你特别愿意为它花钱的东西？为什么？

【打分；理由：题目明确要求1-10分打分，属于平台打分题。后半句"扣分主要扣在哪里"为打分原因追问，建议拆分为独立简答题。】
8. 如果给你现在的生活打个分（1-10分），你会打几分？扣分主要扣在哪里？

【结束画面；理由：模块结束过渡语，不收集受访者回答。】
谢谢你的分享！认识你真的很有意思。接下来我们聊聊你的日常节奏～

### 模块2：我的一天
模块类型：访谈题

【开场白；理由：引导语为模块开头说明，不收集受访者回答。】
接下来，请带我们走进你典型的一天。不用刻意美化，就是最真实的日常。

【简答；理由：需要受访者按时间线还原一天的完整节奏，属于场景还原式开放叙述。】
1. 描述一下你最近典型的一天吧——通常几点起床、几点休息？从早到晚，大概都在做什么？

【简答；理由：需要受访者对比工作日与休息日的差异并具体描述，属于对比式开放叙述。】
2. 工作日和休息日，你的节奏有什么不同？哪些事情只在工作日发生，哪些只在休息日？

【简答；理由：需要受访者描述具体的仪式感时刻与细节，属于场景细节开放叙述。】
3. 在你的日常里，有没有一些固定的"仪式感"时刻——比如早上的某个习惯、下午的某个放松方式，或者睡前的某个动作？具体说说。

【简答；理由：需要受访者描述精力状态的时间分布与主观感受，属于开放叙述。】
4. 你一天中最有状态的时间段是什么时候？最容易分心或疲惫的时间段呢？

【结束画面；理由：模块结束过渡语，不收集受访者回答。】
很好，我们对你的日常有了初步的感觉。接下来聊聊书写这件事～

### 模块3：我和书写
模块类型：访谈题

【开场白；理由：引导语为模块开头说明，不收集受访者回答。】
"书写"这件事，对不同的人来说意义完全不同。我们想听听你的真实感受——不只是用笔写字，而是更广义的"动笔"这件事。

【简答；理由：需要受访者还原最近一次动笔的具体场景（地点、行为、内容），属于场景还原式开放叙述。】
1. 你现在还会动笔写东西吗？在哪些情况下会写？请尽量具体描述最近一次动笔的场景——当时在哪里、在做什么、写了什么。

【简答；理由：需要受访者描述书写场景的纵向变化（增加/减少/消失/新出现四个维度），属于核心变化链路的开放叙述，不可合并或删除。】
2. 和两三年前相比，你动笔的频率和场合有没有变化？哪些场景写得更多了，哪些写得更少了，有没有完全消失的，或者新出现的？

【简答；理由：需要受访者阐述书写不可替代性的具体原因与场景，属于动机探索式开放叙述。】
3. 有没有哪些事情，你觉得"一定要用笔写，不能用手机或电脑替代"？为什么？

【简答；理由：需要受访者描述书写的非功能性价值（情感、认知、表达等），属于深层意义探索的开放叙述。】
4. 除了记录信息，动笔对你还有什么其他意义？比如放松、整理思路、表达自己，或者别的什么？

【简答；理由：需要受访者还原一个具体的愉悦书写场景（状态、内容、工具、感受），属于极致正向体验的场景还原。】
5. 你有没有特别享受的书写时刻？描述一下那个场景——当时的状态、在写什么、用的什么笔、感觉怎么样。

【简答；理由：需要受访者描述书写的负面体验与具体情境，属于极致负向体验的开放叙述。】
6. 反过来，有没有让你觉得"写字好麻烦"或者"不想动笔"的时候？是什么情况？

【结束画面；理由：模块结束过渡语，不收集受访者回答。】
谢谢你聊了这么多关于书写的感受！接下来我们来看看你现在都有哪些笔～

### 模块4：我的笔和文具
模块类型：访谈题

【开场白；理由：引导语为模块开头说明，不收集受访者回答。】
现在，请带我们认识一下你的"笔的世界"。不管多还是少，都欢迎分享。

【简答｜必填图片；理由：题目要求受访者拍摄笔的全家福照片，是后续所有追问的素材基础，属于必填图片的开放叙述任务。】
1. 拍一张你现在拥有的笔的"全家福"吧——笔袋里的、桌上的、抽屉里的，都可以。如果方便，也可以把你的文具收纳区域一起拍进来。

【简答；理由：需要受访者基于全家福照片识别最常用与最喜欢的笔并解释差异原因，属于对比式开放叙述。】
2. 看看这张全家福，哪支笔是你最常用的？哪支是你最喜欢的？这两个答案一样吗？为什么？

【简答；理由：需要受访者描述最近购买行为的具体对象与动机，属于购买决策的开放叙述。】
3. 有没有最近新买的笔？是什么，为什么买它？

【简答；理由：需要受访者描述闲置笔的具体情况与"退休"原因，属于使用变化的开放叙述。】
4. 有没有买了但现在不怎么用的笔？是什么情况让它"退休"了？

【简答；理由：需要受访者描述未满足的购买欲望与犹豫原因，属于潜在需求探索的开放叙述。】
5. 有没有一直想买但还没买的笔？是什么，是什么让你还没下手？

【简答；理由：需要受访者描述购买渠道偏好与决策方式，涉及具体行为过程，属于开放叙述。虽然包含渠道和决策方式两个维度，但适合用自然叙述方式回答，不需要结构化选项。】
6. 你通常在哪里买笔？线上还是线下，哪个平台或店铺？买笔的时候，你一般怎么做决定——冲动购买、货比三家，还是有固定偏好？

【简答；理由：需要受访者描述购买频率、数量与花费，虽涉及数字信息，但DG中适合用自然叙述方式回答，不需要结构化量表。】
7. 你一般一次买几支笔？大概多久买一次？每次大概花多少钱？

【简答；理由：需要受访者描述品牌偏好与具体原因，属于品牌认知的开放叙述。】
8. 你有没有特别喜欢或特别不喜欢的笔的品牌？为什么？

【结束画面；理由：模块结束过渡语，不收集受访者回答。】
你的笔的世界很有意思！接下来我们聊聊你选笔的标准～

### 模块5：我怎么选一支笔
模块类型：访谈题

【开场白；理由：引导语为模块开头说明，不收集受访者回答。】
每个人选笔的逻辑都不一样。我们想深入了解你的标准——不是泛泛的"好用"，而是具体到你真正在意的那些细节。

【简答；理由：需要受访者自由描述"好笔"的具体评价维度与细节标准，属于核心评价标准的开放叙述。】
1. 你觉得一支"好笔"应该是什么样的？请尽量具体——不只是"顺滑""好看"，而是你真正在意的那些细节。

【简答；理由：需要受访者还原极致正向用笔体验的具体场景与感受，是评价标准的核心采集路径，不可删除。】
2. 回想一下你用过的笔里，体验最好的一次是什么情况？当时用的是什么笔，好在哪里？尽量描述那种感觉。

【简答；理由：需要受访者还原极致负向用笔体验的具体场景与问题，是评价标准的核心采集路径，不可删除。】
3. 反过来，有没有让你特别失望或不舒服的用笔体验？是什么情况，哪里出了问题？

【简答；理由：需要受访者描述颜值在选笔中的权重与理想外观的具体想象，属于审美偏好的开放叙述。】
4. 颜值对你选笔有多重要？你理想中的笔在外观上应该是什么样的？

【简答；理由：需要受访者阐述品牌在选笔中的角色与具体含义，属于品牌认知深层探索的开放叙述。】
5. 你会在意笔的品牌吗？品牌对你来说意味着什么——品质保证、身份表达，还是其他？

【简答；理由：需要受访者描述颜值与写感之间的权衡逻辑与具体取舍，属于决策冲突的开放叙述。】
6. 如果一支笔颜值很高但写起来一般，你会买吗？如果写感很好但外观普通呢？

【简答；理由：需要受访者描述笔作为礼物的具体经历与看法，涉及场合、对象、态度等多维度叙述。】
7. 你有没有把笔当作礼物送人，或者收到过笔作为礼物？是什么场合，你怎么看待"笔作为礼物"这件事？

【结束画面；理由：模块结束过渡语，不收集受访者回答。】
你对笔的理解很有深度！最后，我们来聊聊你对未来的笔有什么期待～

### 模块6：我心目中的下一支笔
模块类型：访谈题

【开场白；理由：引导语为模块开头说明，不收集受访者回答。】
这是最后一个模块，也是最有想象空间的一个。基于你之前分享的一切，我们想听听你对"理想中的笔"的期待。

【简答；理由：需要受访者描述最希望被解决的具体书写痛点或场景需求，属于需求优先级的开放叙述。】
1. 如果现在市面上有一支笔，能完美解决你某个具体的书写场景或痛点，你最希望它解决的是什么？

【简答；理由：需要受访者识别现有笔类产品的空白与不足，属于品类机会探索的开放叙述。】
2. 你觉得现在的笔类产品，有没有哪些地方让你觉得"一直没有做好"或者"一直没有人做"？

【简答；理由：需要受访者自由想象一支代表自己的笔的多维度特征，属于产品想象的开放叙述。】
3. 想象一支"最能代表你"的笔——它是什么样的？外观、手感、功能、使用场景，随便说。

【简答；理由：需要受访者描述愿意长期随身携带的笔的具体条件，是国民级大单品潜力的关键判断题，不可删除。】
4. 如果有一支笔，你愿意每天随身携带、用很多年，它需要具备哪些条件？

【简答；理由：需要受访者分享社媒上心动的笔/文具内容及心动原因，属于趋势信号采集的开放叙述，可附截图或链接作为可选补充。】
5. 你有没有在社交媒体上看到过让你心动的笔或文具内容？是什么，为什么心动？如果有截图或链接，欢迎分享。

【简答；理由：需要受访者用一句话或一个词概括笔的个人意义，属于总结性开放叙述。】
6. 最后，用一句话或一个词，说说"笔"对你现在的生活意味着什么。

【结束画面；理由：结束语为全卷最终感谢语，不收集受访者回答。】

## 题型检核摘要

1. **模块1第8题（打分+简答混合）**：当前题面"打几分"与"扣分扣在哪里"合并在一题中。平台打分题仅支持1-10分数值输入，无法同时收集开放文字。建议拆分为两题：一题【打分】收集分值，一题【简答】追问扣分原因。如不拆分，需确认平台是否支持打分题附带文字输入框。

2. **模块1第5题（必填视频 vs 必填图片）**：设计方案标注"必填视频或必填照片"，当前按【简答｜必填视频】标注。待确认：平台是否支持"视频或图片二选一"的必填逻辑？如仅支持单一媒体类型必填，需明确选择视频还是图片。若受访者设备条件不一，建议降级为【简答｜必填图片】以降低完成门槛。

3. **模块4第6-7题（购买行为连续追问）**：第6题（渠道+决策方式）和第7题（频率+数量+金额）均为简答，但题面偏问卷化，连续追问可能让受访者感到负担。已在Wording Handoff中标注，建议wording阶段合并语气或拆成对话式追问。题型本身（简答）无需调整。

4. **全卷题型分布确认**：全卷33题+6段引导语+6段结束语中，简答30题、打分1题、开场白6处、结束画面6处，符合DG以开放叙述为主的设计原则。无单选/多选/排序/AI-bot题型，因本项目为探索性研究，不涉及分支跳转或结构化筛选，题型分布合理。

5. **无中场休息**：全卷6个模块、33道题目，总量较大。建议在模块3与模块4之间（约全卷中段）插入一处【中场休息】，给受访者喘息空间。具体文案交由wording阶段处理。
