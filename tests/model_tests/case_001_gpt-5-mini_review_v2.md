# 题目设计方案

## 1. 项目理解（结论式）
- 项目名称：Mars Gum Occasion Study（口香糖场景与品类破局研究，益达为主要品牌，绿箭作为参照）
- 品类/品牌：口香糖（益达 / 绿箭）
- 目标人群：18-40岁 Urban Strivers（含学生、脑力工作者、体力工作者等分支）
- 商业问题：品类长期下行，需要找到可放大且能迅速招募新购买者的场景与抓手（传播/产品/渠道的组合机会）
- 研究目标：理解真实使用与未使用场景、益达“recenter/好状态”情绪结构、绿箭“餐后/清新”卡点、以及购买/使用链路与习惯养成障碍
- 方法建议：Digital Diary 为主，6 天追踪（覆盖工作日与休息日），辅以后续 IDI（待确认）

## 2. 核心研究问题
1. 在哪些真实场景（功能性+情绪性）消费者会想到并付诸行动去使用“嘴巴解决方案”（口香糖/薄荷/含片/饮料等）？
2. 益达“嚼出好状态 / re-center”的情绪好处在日常场景中是否被理解并能转化为购买/尝试动机？关键触发点是什么？
3. 绿箭“餐后/清新”心智为何难以完全转换为购入行为？餐后场景的购买/使用路径有哪些卡点？
4. 消费者当前的替代方案是什么（哪些品类/产品在抢占场景与心智）？替代方案的优势在哪儿？
5. 购买与使用链路中有哪些可操作的产品、包装或渠道抓手能显著降低准入门槛并提高复购？

板块1：关于我
引导语：先介绍一下你自己，帮助我们理解你平时的生活节奏和压力来源，不需要太专业，用你自己的话说就好。
题目：
1. 请用两三句话介绍你自己（年龄、职业/在读学校、住处类型：合租/独居/与家人同住）。
2. 你觉得最近一个月你生活/工作的节奏是怎样的？（比如忙碌、波动、稳定、熬夜/朝九晚五等）
3. 最近一个月里让你觉得“状态不好/不自在”的主要原因是什么？请举一个最近发生的具体小例子来说明那种感觉。
4. 在日常里，你通常用哪几种简单方式来“调节情绪/恢复状态”（可以是喝饮料、吃东西、听歌、走动、呼吸等，按重要性列出最多 5 个，并简短说明为什么选它们）。
5. 如果用一句话描述“你觉得的好状态（自在/精神饱满）”，你会怎么说？描述一个你觉得典型的“好状态”瞬间。
结束语：谢谢你！接下来我们会让你把“典型一天”画出来，帮助我们把这些状态放到时间里看。

板块2：我典型的一天
引导语：请把你的一天用“关键时间段和状态”说给我们听，不需要写很长，重点是那些你能明显感到能量/情绪变化的时刻。
题目：
1. 请从早到晚列出你一个典型工作日的主要时间段与在那段时间你的主要活动（例如：7:30-8:30 吃早饭+通勤；9:00-12:00 工作/上课），每段用一行写清时间和活动。
2. 在这一天里，你什么时候最容易感到“精神下滑”或“注意力不集中”？请具体描述至少一个场景（时间、地点、当时在做什么、当时感受）。
3. 有没有你会有“准备上场/重要时刻前”的例子（例如面试、汇报、见朋友等）？请描述一次最近的例子和当时你为了准备/缓解紧张做了什么。
4. 在休息日（周末/节假日），你的日常和工作日有什么不同？请写两处最明显的差别。
5. 想一想上周，你有没有在出行/开车/学习/开会等场景里做过一个短小的“仪式/动作”（比如喝一杯提神饮、吃点心、做深呼吸）来让自己“回到状态”？描述一次并写出当时是否有效及原因。
结束语：接下来我们会问到你日常里吃喝以外的那些小东西，别担心，我们不会马上问你为什么不吃口香糖，会先听你讲你的“吃吃喝喝”习惯。

板块3：我正餐以外的吃吃喝喝
引导语：我们把“正餐以外、不以填饱为目的”的那些吃喝统称为“随手小吃/小饮”，请帮我们画出你最近一周里这些小东西的地图。
题目：
1. 请列出你最近一周里至少 5 种你在正餐之外会吃或喝的东西（例如：茶饮、咖啡、零食、薄荷糖、口香糖、坚果、小面包等），并在每种后面写上你通常在什么场合吃它（例如：通勤/午后下午茶/加班）。
2. 在家里或包里，你通常会放哪些“随手小吃/小饮”？请拍一张你包里或家中常备的这些东西的照片并上传（如果不方便拍，写明放在哪儿）。
3. 对这些替代品，你觉得它们能帮你解决什么“当下的问题”（例如：提神、解渴、打发时间、缓解尴尬、吃完换口味等），请分别写一句话说明。
4. 最近 2 年内，你有没有新增或减少的“随手小吃/小饮”习惯？举一个新增的例子，说明是什么促成的（方便/好吃/被朋友带动/广告等）。
5. 当需要“立刻解决嘴里不舒服/口气”或“提振精神”时，你通常第一反应会做什么？（请按优先顺序写 1-3 件事，并解释为什么）
结束语：谢谢你的回答。下面进入我们的日记模块——我们会请你在接下来 6 天中记录发生的具体时刻。

板块4：我的6天日记（每日模板）
引导语：接下来 6 天，请按下面的模板记录你当天发生的与“嘴巴/提神/情绪调节”相关的真实时刻（包括你主动去吃某样东西、或本想吃但没有吃的时刻）。每次记录尽量在事件发生后 1 小时内提交。可以用文字、语音或拍照/短视频来记录，尽量真实自然。
题目（每日需回答；每天重复）：
1. 今天第 X 次记录：请给这次事件取一个简短标题（例如“中午会议前紧张/加班疲劳想提神/开车半路困了”）。
2. 时间与地点：这次事情发生在什么时候？在哪里？
3. 当时在做什么？（例如：上班、在地铁、准备汇报、开车等）
4. 触发原因：是什么让你想要做点什么来“调整状态/清新口气/提神/打发时间”等？请尽量具体描述当时的感受。
5. 你当时采取了什么行动？（写出你做的第一件事到最后一件事；如果吃了某样东西，请写出品牌/品类和你吃了多少）
6. 在做选择时，你还考虑过哪些替代方案？为什么最终选择了现在的这个（包括选择“不做”的情况，请说明原因）？
7. 如果这次有吃口香糖，请描述：什么品牌（或你记不得品牌请写样貌）、什么时候开始嚼、嚼了多久、嚼感如何（强/温和/清凉/甜/没味道）、吃完觉得状态有什么变化（描述细节）。
8. 如果没有吃口香糖但用了其他替代品，请描述替代品带来的即时效果和为什么没有选择口香糖（没想到/不方便/嫌麻烦/不喜欢等）。
9. 效果评估：这一动作/产品对你的问题帮助程度，请打分 1-5（1 完全无效，5 非常有效），并写一句说明为什么给这个分数。
10. 上传（任选其一）：当时的环境照片/商品照片/购买凭证截图/1分钟视频或语音说明（说明该素材能帮助我们理解什么）。
结束语（每日）：谢谢你的记录！如果还有未写完的补充想法可以在下方写一句话。

板块5：购物任务（安排在第4-5天）
引导语：我们想看到你在自然状态下的购买路径。请在第4或第5天任选一日完成下面的购物任务（线上或线下任一渠道都可），尽量像平时那样自然购物。若你无法外出，请做线上版并尽量按自然路径操作并截屏。
题目：
1. 请拍一张你将要去的线下货架或线上页面的截图/照片（要求能看到口香糖/薄荷等集中陈列位置及周边品类）。
2. 货架观察：这些口香糖放在收银台附近、零食区还是结账等待区？周边有哪些替代品（薄荷糖、含片、能量糖、口腔护理产品等）？
3. 有无促销/组合装/小包装？价格是多少（写出见到的价位）？
4. 你有没有拿起或考虑拿起益达或绿箭的产品？请拍下你拿起的商品并录一段 30-60 秒的视频说明你当时的想法（比如：为什么拿/没拿、被什么信息吸引或劝退）。
5. 最终是否购买？如购买请上传小票或支付截图并写出购买原因；如未购买请写出最主要的犹豫点（例如价格、包装、没有合适口味、觉得不需要等）。
6. 若你没有自然打算购买，我们也欢迎你在这个环节模拟一次“如果现在需要我会怎么买”的流程，并说明你会先看什么、选哪个渠道、为什么。
结束语：购物任务完成后请休息一下并在接下来两天内把你选择的“实际吃一次”记录在日记中（见第4板块）。

板块6：嘴巴习惯养成
引导语：我们想理解某些“小东西”如何从偶尔使用变成日常习惯。请结合你个人经历描述一个长期养成或中断的例子。
题目：
1. 请选出两种你曾经有过“长期/固定”使用习惯的正餐外小吃（可以是口香糖，也可以是其他零食或饮料），并分别写出你开始习惯的第一个记忆（什么时候、谁、为什么开始）。
2. 促成习惯的关键因素是什么？（例如：便捷的包装、被朋友带动、口味/功效明显、价格合适、放在显眼位置等，请列 2-3 个原因并举例说明）
3. 有没有哪一次你一周甚至更长时间没有继续使用某个你原本常用的产品？是什么原因让这个习惯中断？之后又是怎样恢复或彻底放弃的？
4. 对于口香糖这类产品，哪些具体机制最容易把“偶尔尝试”转成“我会主动买来放”的习惯？（比如：小包装便于试用、提醒性陈列、搭配场景、社交推荐等，写出你认为最有效的两点并说明）
5. 如果让你设计一个简单的“拉新/复购”小把戏（不需要很专业），你会怎么做来促使你自己或身边人更常去买口香糖？一句话描述你的点子。
结束语：很好！最后我们想听听你对品牌和理想口香糖的期待。

板块7：我期待的品牌与产品
引导语：最后请放空自己，想象一下理想的“口香糖/嘴巴解决方案”。我们也会问你对已知品牌（益达、绿箭）的印象，但你可以非常随意地写下第一感觉。
题目：
1. 想象一个能够“立刻让我回到好状态/提振精神/缓解紧张”的口香糖，它应该有哪些关键特性？（请按优先级列 3 条，并简单说明）
2. 想象一个用于“餐后清新”的理想小产品（可以是口香糖/含片/喷雾等），它应该怎样设计才能让你在吃饭后真的去买来用？写 2-3 点要素（例如：便于携带、可即时获取、味道持久、价格合理等）。
3. 请分别用 1-2 句话写出你对“益达”和“绿箭”这两个品牌的第一印象（如果你不熟悉可以写“没太注意过”）。（例如：益达让我觉得…；绿箭让我觉得…）
4. 对于益达正在推的“嚼出好状态/recenter”这个方向，你觉得怎样的真实场景或广告信息最能让你产生试用冲动？请写出一个你觉得会有效的短情境或一句话（不需要专业）。
5. 对于绿箭想更好把“餐后心智”转化为购买，你觉得在门店/外卖/餐饮场景里哪个点最重要去优化？写一条你认为最关键的可执行点。
6. 如果现在让你在包装、规格或口味上提一个具体改进（任何你愿意的想法），写一句话说明会是什么并说明为什么你会买。
结束语：非常感谢你完整地参与这 6 天的记录与思考！如果你愿意，最后可以录一段 30-60 秒的语音，总结这次体验中的最重要一个感受（例如：你发现了什么、你觉得什么最关键）。

内部提示（仅供研究员/项目团队）：日记模块应区分“功能性场景”和“情绪性场景”；购物任务优先线下拍照+短视频，无法线下者提供 O2O 截图与操作视频替代。鼓励受访者使用语音或短视频以降低文字负担。

## 5. Agent 检核摘要
- 已对照商业问题与 Proposal 检核：题目覆盖生活背景、场景图谱、6 天即时链路（trigger→action→alternatives→choice logic→体验→评价）、购物路径与习惯养成，能支撑益达与绿箭的差异化机会发现。
- 主要风险1：受访者负担较重（6 天、购物任务、拍视频），可能导致弃测或表面化回答；控制：鼓励语音/短视频、照片替代文字，日记题尽量短且允许跳过非必要题（由研究员在平台设置）。
- 主要风险2：未明确样本/城市分层可能影响场景代表性；控制：建议在招募时按工作/学生/司机等场景分配配额，并在确认点核实。
- 主要风险3：如果没有提供产品样本，吃用体验和品牌主张的测试真实性可能受限；控制：在确认环节询问是否由客户寄样或允许受访者自购并上传凭证。
- 已做的质量控制：问题顺序避免过早暴露品牌/研究意图；品牌评价放在最后以减少对日记行为的干扰；购物任务安排在中段以先收集自然使用证据。
