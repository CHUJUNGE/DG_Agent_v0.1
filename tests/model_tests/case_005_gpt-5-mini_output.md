# 题目设计方案

## 1. 项目理解（5 条结论）
- 项目名称：协助 KACO 打造国民级笔类大单品（MindSight）
- 核心目标：识别未来3年具备“国民级”潜力的笔类/新品方向，支撑品牌长期增长
- 关键研究方向：理解目标人群生活方式与书写场景变化、书写价值与习惯、笔的选择逻辑与未满足需求
- 客户已有认知：笔类为非刚需且被数字化分流，市场新品快速迭代、圈层驱动与审美/把玩属性显著
- 目前缺口（需确认）：目标细分人群范围、日记记录天数与是否安排后续 IDI / 刺激物测试

## 2. 核心研究问题（3-6 个）
1. 用户在真实生活中何时、因为什么场景会选择“拿笔”而非用数字工具？这些场景的触发与价值是什么？  
2. 现在笔在用户生活中承担哪些角色（功能性、情绪/表达、把玩/收藏、社交符号等），不同角色的关键驱动因素是什么？  
3. 用户现有的选择逻辑与替代方案是什么（如何挑笔、在哪买、为何换/弃用、会为哪类特性付钱）？  
4. 哪些未满足的体验或愿望可以被转化为具备广泛吸引力且可放大的产品机会（产品功能、形态、情绪价值、生态）？  
5. 从使用到购买的路径与关键卡点有哪些（发现、尝试、购买、重复/中断、送礼）？

## 3. 模块结构总览

| 模块 | 模块目的 | 对应研究问题 | 保留/合并/定制理由 |
|---|---:|---|---|
| 模块1：关于我（基础画像） | 了解个体生活底色与身份，作为后续场景理解背景 | 全部（尤其 Q1、Q2） | 保留（必须），使用固定开场三题 + 少量补充，避免过早谈笔 |
| 模块2：典型的一天（routine） | 开放描画一天中可能触发书写的时间、状态与差异（工作日/周末） | Q1、Q2 | 保留，帮助构建场景脉络以解释书写行为 |
| 模块3：书写场景与行为图谱 | 系统梳理各类书写场景、替代方案与工具选择逻辑 | Q1、Q2、Q3 | 保留并具体化为“回顾性场景样本”而非抽表格 |
| 模块4：多日日记（实时记录） | 捕捉真实发生的书写事件、触发、选择与感受（建议日记天数需确认） | Q1、Q2、Q5 | 保留为主干，日记题模版化，鼓励多媒体补充 |
| 模块5：购物与收藏任务 | 还原购买/寻找过程、犹豫点与决策理由 | Q3、Q5 | 保留购物/vlog 任务，重在自然记录而非货架审计 |
| 模块6：习惯养成、变化与阻碍 | 追问为何形成/改变使用习惯以及中断原因 | Q2、Q4 | 保留，用于挖掘可触达的长期使用驱动 |
| 模块7：理想笔与机会想象 | 收集开放的产品期待与概念性想法，作为创新方向输入 | Q4 | 保留末段收束，不提前暴露品牌或具体技术 |

（注：若后续安排 IDI/原型测试，模块7 可改为刺激物反应；请在确认点反馈）

## 4. 详细题目设计

### 模块1：关于我（基础画像）
引导语：首先想先认识一下你，没关系不用正式，随便聊聊就好。可以用文字、照片或短视频来表达你自己。
题目：
1. 首先，让我们多了解一下你吧！色彩、MBTI、星座、关键词......可以用任何你觉得能代表自己的方式来介绍自己！欢迎多多分享照片，向我们展示真实的你～
2. 和我们聊聊你的学业/工作吧！包括你所在的专业/行业、每天具体要做的事情（如果你有副业，也介绍一下吧！）
3. 可以拍一段小视频，带我们“云参观”一下你的居住空间吗？（比如宿舍/家里的整体布局、你每天待得最久的地方，你最用心打理的一个角落）
4. 请简单说说你与“笔/文具”的关系：你平时会保留多少支笔？会收集或更换笔吗？一般笔放在哪里？（可以用照片补充你的笔筒/收纳）
5. 当你想到“我的风格”或“我喜欢的东西”时，有哪些审美或小物是你愿意展示的？（例如：手帐、速写、贴纸、收藏等，随意列举）
结束语：太棒了，了解这些会帮助我们把后续的问题放在更接近你生活的角度。接下来的题会聊聊你的一天和使用笔的场景。

（内部提示：前三题为固定开场；第4题用于快速判断是否为笔爱好者/收藏者）

---

### 模块2：典型的一天
引导语：请描画一下你典型的一天吧！不用写时间表，讲讲从起床到睡前你通常在做什么，什么时候会用到笔，心情或状态有没有变化。
题目：
1. 从早到晚，你的一天一般怎么过？哪些时刻你会主动拿笔？那时你通常在做什么、和谁在一起、处于什么状态？
2. 在学习/工作过程中，有没有固定需要用笔的环节（例如记笔记、草稿、签字、手账）？这些环节对你现在来说重要吗，为什么？
3. 工作日和休息日你使用笔的场景有何不同？能描述一个最近在非工作/学习场景下用笔的瞬间吗？当时发生了什么？
4. 请上传或拍一张你常用写作/办公空间的照片（若方便），并简单说说这张图最能体现你的哪种使用场景。
结束语：谢谢你的描绘！接下来我们想请你回忆一些具体的使用时刻。

（内部提示：重点收集日常触发点与情绪/状态变化）

---

### 模块3：书写场景与行为图谱
引导语：回想最近几次你主动拿起笔的情境，挑两到三次最能代表你使用习惯的，描述给我们听。
题目：
1. 请挑一到两个最近的例子，描述当时发生了什么（例如：临时记录灵感、写信、做手账、勾线、签合同、课堂笔记等）。当时你在哪里，为什么要用笔？
2. 在那个场景里，你用了哪种笔？为什么选择它（手感、颜值、墨色、方便携带、习惯、价格、别人推荐等）？
3. 当时你是否考虑过不用笔而用手机/电脑/平板？如果有，为什么最终还是选择了笔（或选择了数字工具）？
4. 那次使用后你对笔的感受如何？有没有希望它改进的地方或当时的“卡点”？
5. 你认为在你的生活中，笔更多承担的是哪类价值？（实用记录 / 表达美感 / 放松/把玩 / 社交展示 / 礼物象征 等，随意描述）
结束语：谢谢你的具体回忆，这些片段对我们理解场景非常有帮助。

---

### 模块4：多日日记（每日记录模版）
引导语：接下来请在接下来的几天里，按下面的模版记录你有关“拿笔/用笔”的真实瞬间。可以用文字、语音或短视频记录，如果方便也可以拍下当时的笔或场景。我们建议记录多天以覆盖工作日与休息日（具体天数请确认）。
题目（单日记录模版）：
1. 今天有没有哪个时刻你主动拿笔？请描述当时发生的事、你当时的状态、地点与陪伴的人。
2. 这次你用了哪支笔？能拍张笔的近照或写字的照片来补充说明吗（如果方便）？
3. 当时你为什么选择用这支笔？有没有什么特别的感觉（写起来顺手/好看/能表达心情/只是凑巧拿到）？
4. 这次使用后你的感受有无变化（例如更专注、放松、觉得更有仪式感、只是完成任务等）？
5. 如果当天没有用笔，也请简单说说原因（全部用数字工具、忘带/找不到、当时不想写等）。
结束语：谢谢你今天的记录。请尽量在接下来的几天里继续用同样的模版记录，形式随意，我们更看重真实片段。

（内部提示：将日记模版保持轻量，鼓励语音/视频以减少文字负担；日记天数建议 4-7 天，待确认）

---

### 模块5：购物与收藏任务
引导语：把一次真实的购买或寻找笔的过程想象成你自己的小 vlog，尽量自然记录你从想到买到决定的过程。可以是线上，也可以是线下。
题目：
1. 回想最近一次你买笔或挑选笔的经历，简单讲讲当时怎么想起要买/看笔的（计划内还是偶发）？
2. 你是在哪里找/看/比的（实体店/电商/社媒/朋友推荐等）？当时你关注了哪些信息或细节？
3. 讲讲你做决定的关键点：是什么让你最终选了这支笔（或放弃某支）？有没有哪个瞬间你觉得犹豫或卡住？
4. 购买后你对这次购物体验有什么感觉（满意、后悔、还会买类似的、只为收藏等）？如果方便可以拍下购买页面或实体包装的一张照片。
5. 你给朋友/家人买笔的思路通常是什么？会当作礼物买吗？为什么？
结束语：很棒，购物过程里的选择逻辑能帮助我们理解市场入口和卡点。

---

### 模块6：习惯养成、变化与阻碍
引导语：我们想了解一些长期的使用变化与背后的原因，不用很学术，按你的生活故事讲就好。
题目：
1. 有没有一段时间你开始特别在意某类笔（例如为了手账、为了绘图、为了签字显档次等）？那段时间是什么触发的？
2. 当你曾经常用但后来减少或停用某支笔/某类笔时，主要原因是什么？（例如：坏了/用完/不顺手/换潮流/数字化）
3. 有哪些因素会让你持续回购或长期使用同一款笔？试着用生活中的例子说明。
4. 如果要把“持续使用一支笔”变成习惯，你认为最关键要解决的是什么？
结束语：谢谢你分享长期的变化与原因，这些能告诉我们哪些设计更可能被长期接受。

---

### 模块7：理想笔与机会想象
引导语：最后请自由想象一支你认为“很可能成为人人都想要那种笔”的产品。可以天马行空，也可以脚踏实地，说出你的想法。
题目：
1. 如果让你设计一支“国民级”的笔，你会把它做成什么样？请从外观、手感、功能、使用场景、价格、购买渠道或配套生态任意描述。
2. 这支笔在生活中能为你带来什么不同（比如让你更愿意写、让写字更有仪式感、或更方便携带……）？
3. 你希望这支笔如何被发现和传播？（朋友推荐、社媒种草、校园活动、礼物、IP 联名等）
4. 如果你看到一个笔的概念图或短视频，哪类信息最能让你产生兴趣并想试用？（外观、材质、功能演示、短体验视频等，随意描述）
结束语：感谢你的想象与建议！这些开放的看法会直接转化为可尝试的产品方向。

（内部提示：如后续有原型/概念刺激物，建议把此模块改为实测反应并安排 IDI 深入理解）

## 5. Agent 检核摘要
- 已核对 Proposal 中的核心目标：方案覆盖“生活方式—场景—选择逻辑—机会想象”的完整链路，题面按日记优先、开放为主设计。  
- 主要风险：目标人群范围与日记天数未明确，会影响样本与负担控制；是否需后续 IDI/刺激物未确定，将影响模块7的作法。  
- 负担控制：日记模版保持轻量，鼓励语音/照片/短视频以降低文字门槛；购物任务用自然 vlog 口吻避免货架审计感觉。  
- 刺激物与深层认知：本版不包含概念刺激测验；若需要检验具体概念，建议安排 IDI 或在日记末端加入短刺激并做二次探讨。  
- 样本与筛选：方案预置能区分“高频使用者/收藏者/偶用者”，建议在招募与初筛里明确（在实际执行前补充筛选题）。

## 6. 需要确认的问题（最多 3 个）
1. 目标人群请确认本轮聚焦人群：以 Proposal 建议的“05/10后学生 + 年轻白领”为主样本，还是需要更广/更具体的分层（例如仅学生、仅职场、或按圈层/兴趣分）？  
2. 日记记录时长请确认：建议 4–7 天（含至少一个工作日与一个休息日），请确认最终天数与每日提交频率（影响题量与激励设计）。  
3. 是否计划在研究流程中安排后续 IDI 或把具体产品概念/原型作为刺激物进行测试？（若是，模块7 需改写为刺激反应模块并保留深访承接）

暂无其它必须确认的问题；可在确认以上三点后，我将据此调整题量与刺激物/IDI 安排并输出修订版。
