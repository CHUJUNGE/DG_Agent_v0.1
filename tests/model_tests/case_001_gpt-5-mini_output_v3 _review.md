# 题目设计方案

## 1. 项目理解（精简结论）
- 项目名称：Mars Gum Occasion Study（口香糖场景机会研究，重点益达 + 绿箭）
- 品类/品牌：口香糖；主要关注益达（Extra）与绿箭（Doublemint）
- 目标人群：18-40 Urban Strivers（包含学生/脑力工作者/外出工作者等）
- 关键商业诉求：加速生意恢复——更快地 recruit live buyers / new buyers，通过场景/产品/沟通等抓手把心智转化为购买
- 已有沉淀与方向：客户已有大量品类与品牌研究，益达定位“recenter/好状态”，绿箭定位“餐后/清新”；研究应在既有认知上进一步具体化场景、动机与购买卡点
- 方法与节奏建议：Digital Diary（6天，包含工作日与休息日），搭配后续深访（如有）用于深入解释行为背后动机

## 2. 核心研究问题
1. 在哪些真实场景下（功能与情绪），口香糖仍然有机会进入或成为首选解决方案？  
2. 目标人群在“需要调节状态/提振精神/餐后清新”等时刻当前怎么做？口香糖在替代路径中扮演什么角色/为何未能成为首选？  
3. 益达的“嚼出自在/好状态”定位在生活中的可感知触点与沟通空白在哪里？如何让认知更具体并推动购买？  
4. 绿箭的餐后心智为何未能稳定转化为购买习惯？购买与使用链路（购买前-中-后）有哪些卡点？  
5. 在产品、包装、渠道或沟通层面存在哪些可放大的机会点，能较快促进试买与复购？

## 3. 模块结构总览
| 模块 | 模块目的（简述） | 对应核心研究问题 | 保留/定制说明 |
|---|---:|---|---|
| 模块1：关于我（基础画像） | 建立生活底色、压力源、价值与日常节奏 | 全面支持 Q1、Q2 | 必保；前3题为固定开场（规则库） |
| 模块2：我典型的一天（routine） | 描画一天中状态波动与“off-center/re-center”时刻 | Q1、Q2、Q3 | 必保，放在早段 |
| 模块3：我正餐以外的吃吃喝喝（图谱） | 下载广义“嘴巴相关”解决方案图谱（包括薄荷糖、饮料、口腔用品等） | Q1、Q2、Q5 | 必保，避免直接提口香糖，先画替代图谱 |
| 模块4：我的6天日记（每日记录模板 + 指定日任务） | 捕捉即时触发、替代方案、选择逻辑、体验后效果 | 全题核心（Q1-Q5） | 必保；在第4日布置购物任务，第5-6日包含 eating experience |
| 模块5：购物任务（自然购买/回溯） | 重现购买链路与卡点（前/中/后） | Q2、Q4、Q5 | 必保；以自然 vlog 式记录为主 |
| 模块6：嘴巴习惯养成（Habit） | 追溯习惯形成/中断机制与可借鉴的养成触点 | Q2、Q5 | 必保；拓展至其他成功养成的食品/习惯作为启发 |
| 模块7：我期待的品牌与产品（品牌印象与期待） | 最后进入益达/绿箭印象、品牌可联想画面与理想产品要素 | Q3、Q4、Q5 | 放最后以免提前影响日记/行为记录 |

（注：模块目的与对应问题仅用于设计逻辑说明；详细题面见下文）

## 4. 详细题目设计
建议：
1. 模块一的引导语以及结束语是可以的，但是其他模块的结束语不行，首先不能直接出现品类以及该模块的目的给受访者，应该是更加口水词的内容，请自行对照final dg去改善
2.购物任务或者使用任务这类型题目一般是品牌直接给出的想法，需要说出购买/使用的品牌名字，比如这处你说的口香糖，但是根据背景资料，你是知道是益达和绿箭，那么就该直接露出品牌。



模块1：关于我 / About Me  
引导语：首先让我们轻松认识一下你，没必要很正式，按真实的自己来就好。可以用文字、照片或小视频补充你觉得能代表自己的内容。  
题目：  
1. 首先，让我们多了解一下你吧！色彩、MBTI、星座、关键词......可以用任何你觉得能代表自己的方式来介绍自己！欢迎多多分享照片，向我们展示真实的你～  
2. 和我们聊聊你的学业/工作吧！包括你所在的专业/行业、每天具体要做的事情（如果你有副业，也介绍一下吧！）  
3. 可以拍一段小视频，带我们“云参观”一下你的居住空间吗？（比如宿舍/家里的整体布局、你每天待得最久的地方，你最用心打理的一个角落）  
4. 请简单说说你最近一个月里，工作/学习/生活上主要的压力或让你“不太自在”的事是什么？这些事情通常什么时候发生？（如果方便，可以配一张能代表当下心情的照片）  
5. 平常你最喜欢的放松方式、打气或调整状态的小动作是什么？（比如吃某种零食、短暂散步、听歌、和朋友聊两句等）  
结束语：谢谢你！接下来我们把视角拉到一天中的时间轴，跟我们聊聊你的典型一天吧～

模块2：我典型的一天 / My Typical Day  
引导语：请描画一下你典型的一天，从早到晚，一天里做的事、心情和能量变化都可以写出来。不需要写得很正式，像跟朋友聊天一样就好。可以用文字、语音或几张照片来呈现。  
题目：  
1. 请从早到晚描述你典型的一天，大致什么时候起床、什么时候最忙、什么时候结束一天？（可以用几句画面感强的描述）  
2. 在这一天里，你会在哪些时刻觉得状态不太在、需要“缓一缓”或需要“提振一下”？举一个最近发生的具体例子，描述当时你在做什么、在哪里、和谁在一起。  
3. 当你处于上题那种“状态不太在/需要提振”的时刻，你通常会怎么做来调整？请描述你实际采取过的一次完整流程（包括想到的办法、选择的替代品、最后效果）。  
4. 你的一天里有没有固定的“短暂休息/补给”时刻（比如课间、下午茶、通勤中间等）？这些时刻你通常会带什么或买什么来应对？  
结束语：好的，很清楚了。下一部分我们想把视角放到“嘴巴相关”的各种解决方案，不只口香糖，先把所有可能的替代方案都画出来。

模块3：我正餐以外的吃吃喝喝 / Everyday Mouth-Related Items  
引导语：这里的“正餐以外的吃吃喝喝”指的是不以填饱肚子为主目的的那些东西——小零食、薄荷糖、含片、能量软糖、能量饮、口喷、口腔清新剂等。请按你真实生活来写，不用担心种类多。  
题目：  
1. 回想一下你家/包里/工作位经常囤或随手拿的那些“正餐以外”的物品都有哪些？请写出你这周内至少用过或看到过的若干种（可以用逐条短句列出，文字+可选照片）。  
2. 当你在想要“提神/换脑/缓解紧张/餐后清新/打发无聊”等目的时，你会优先想到哪些具体东西？请说说每种东西你选择它的原因（方便/便宜/效果/习惯/别人推荐等）。  
3. 在这些替代品中，有没有让你会“想到并实际去买/去拿”的？有没有只是“想到但没去做”的？请举一两例说明当时的情境与原因。  
4. 你能想到的与“口香糖”类似或替代口香糖的所有东西里，哪个场景下它们最常被使用？（比如上课走神时、面试前、餐后、开车长途、熬夜工作等；可写多个场景）  
结束语：谢谢你把图谱画出来了，接下来我们会在未来几天里让你记录真实的日常时刻，尽量把刚才提到的东西自然带进日记里。

模块4：我的 6 天日记 / 6-Day Diary  
引导语：接下来是我们6天的日记记录。每一天请在当天或当天晚上回到这里，按真实发生的时刻记录。文字、语音、照片或短视频都可以用来补充，不用写长文，按你平常的表达就好。我们会在第4天安排一次购物任务，第5-6天会邀请你记录一次食用/体验（如果你平常不吃口香糖，请如实说明；若愿意尝试，后面会说明如何做）。  
题目（每日模板，Day1–Day6 均使用，另有指定日任务见下）：  
1. 今天有没有发生让你想调整嘴巴状态或想改变自己状态的时刻？（比如想清新口气、提提神、缓解紧张、打发无聊等）如果有，请选一个最清晰的时刻来描述：发生时间、地点、当时在做什么、和谁在一起。  
2. 那个时刻你当时做了什么来应对？（比如吃了某样东西、喝了饮料、深呼吸、走一走、什么都没做等）请描述你完整的决策过程（当时你想到的备选项、最终选了哪个、为什么）。如果你想补充可以上传一张照片或一段短语音说明。  
3. 在你采取的做法里，口香糖有没有出现在你的备选清单里？如果有，是什么阻止或促使你去吃/拿它？如果没有，是什么替代了它？请尽量说清楚当时的考虑。  
4. 结果如何？（你的状态有没有变化、持续多久、有没有影响后续的事情）  
5. 今天有没有因为嘴巴/口气/状态问题需要购买或取用什么东西？如果有，请简述你是如何获取（包里现有、同事/朋友给、路上买、网购等），并上传相关照片/截图（如有）。  
每天结束语：谢谢你的当天记录！如果方便，睡前可以再录一段简短语音说说今天这类时刻对你来说重要不重要，以及有没有想过在未来会怎么处理类似情况。

指定日任务（嵌入在第4、5、6天的日记中）：
- 第4天 — 购物任务：如果近期内你有计划外出或当天下单购买任何“正餐以外”的嘴巴相关产品（口香糖/薄荷糖/含片/口喷或类似），请尽量用手机录一段短视频，像拍生活 vlog 那样记录整个流程（出门/在店里/下单/付款/回到位放置）；若当天没有购买，请回忆并文字/语音回溯最近一次购买经历（何时、为什么买、如何决定）。（如果不方便录视频，也可以用文字或语音详述）  
- 第5天 — 吃用体验 A：请在当天记录一次你自愿的“吃用体验”（可以是你平常吃的任何嘴巴相关产品，也可以选择尝试一款口香糖，按你愿意的方式），记录：为什么在这时想吃、过程（味道/嚼感/持续时间/周围有人否）、吃完后效果与当下感受。可用短视频或语音补充。  
- 第6天 — 吃用体验 B（对比/反思）：回顾这6天里最有代表性的两次“嘴巴相关”体验（可以包括第5天），比较它们哪个更能解决你的需求，为什么。最后写一句话评价：如果有一个理想的“嚼出好状态”的产品，它最应该做到什么？  
结束语：感谢你连续6天的记录！你的这些真实场景和体验对我们非常重要。

内部提示（仅供研究员）：第4天优先观察购买中途决策与货架位置/可见度，第5-6天用于真实食用体验记录与前后状态变化。

模块5：购物任务 / Shopping Task  
引导语：我们想更真实地看到你从“想到买”到“完成购买/放弃”的全过程。请把最近一次购买或第4天购物任务的过程尽量用视频+语音记录，像给朋友拍一个短 vlog。若你不购买，也请回忆并按下面问题描述一遍最近一次完整购买体验。  
题目：  
1. 请描述（或上传视频）你这次购买的完整过程：从最初想买的瞬间、出发、在店里或App上如何找到、比较过哪些替代品、最终为什么选择了它。  
2. 在购买过程中你遇到过哪些犹豫或障碍？（例如：没找到想要的口味、价格/包装因素、不好拿、不方便在当时使用等）你当时是如何处理这些犹豫的？  
3. 购买完成后你把产品放在哪儿？你一般会如何安排它以方便下次使用（随身/置于办公桌/家里冰箱/包里等）？为什么这样放置？  
4. 如果你曾经看到过口香糖在收银台或某个位置促销，你会因此而增加购买可能性吗？请描述一次你因此购买或没购买的例子。  
结束语：谢谢，这些购买细节帮助我们理解渠道与现场卡点。

内部提示（仅供研究员）：观察购买决策的触点（可见性/价格/包装/小包装便利性）以及是否有因“准入成本”太高而放弃。

模块6：嘴巴习惯养成 / Mouth Habit Formation  
引导语：我们想了解一些成功/失败的习惯养成经验，这可以帮助找到让口香糖进入日常的路径。请回忆两类产品经历：一个是你最近1-2年里新养成的“小习惯”（比如开始每天某时喝一杯、常备某零食等），另一个是你一直以来长期保留的“老朋友”式产品。  
题目：  
1. 请选择一个你最近养成的新习惯，简述它是什么、是什么契机让你开始（一个事件/别人推荐/促销/情境变化等）、当时你是如何坚持下来的。  
2. 这个新习惯在形成过程中有没有遇到断档或放弃的时刻？是什么原因让它持续/中断？有没有什么外部工具或触发（包装、提醒、社交、可得性）帮助你坚持？  
3. 请选择一个你长期保留的产品习惯，回顾最早一次开始使用的情境，它为什么会变成“必须带”的东西？  
4. 综合上面两例，谈谈对你来说一个新消费习惯需要哪些关键条件才会从“想到”变成“每天/常带”的真实习惯？  
结束语：好的，接下来我们进入最后一节，聊聊品牌印象与你期待中的理想产品。

模块7：我期待的品牌与产品 / Brands & Products I Look Forward To  
引导语：这是最后一部分，放在最后是为了不影响你前面几天的自然记录。现在我们想了解你对“嚼出好状态/餐后清新”类主张的直觉画面，以及对益达和绿箭的印象（可以坦诚表达真实感受）。  
题目：  
1. 当你看到“嚼一口能让自己回到更好状态/帮你缓解卡住的思路”这样的画面时，你脑海里会浮现出什么样的生活场景或人物？请用几句话或一幅你自己画的草图/照片来表达。  
2. 请分别说说你对“益达（Extra）”和“绿箭（Doublemint）”这两个名字或品牌的第一印象是什么？（可以从包装、在哪看到过、你认为它适合什么场景、你是否会主动去买等角度写）  
3. 想象一个理想的口香糖产品，它需要满足哪些要素才能让你在“需要调节状态/餐后/提神/社交”时优先选择它？请把最重要的两三条写出来（例如：便携、口感持续、味道变化、价格、社交认同感等）。  
4. 如果品牌要让“嚼出自在/好状态”这个主张更具体化，你觉得应该以什么样的沟通或产品改动更容易让你去试买并留下来？举两个你能接受的具体做法（可以是沟通场景、包装、试用小样、渠道合作等）。  
结束语：谢谢你完整地走完了这次6天记录！如果你还有什么想补充的生活小片段、照片或视频，很欢迎你在这里上传，任何真实的细节都很有价值。

内部提示（仅供研究员）：最后阶段允许明确提及益达与绿箭以收敛品牌印象与可执行建议。

## 5. Agent 检核摘要
- 已对照 Brief/Proposal 与 Case_001 规则：模块覆盖生活画像、场景图谱、6天日记、购物任务、习惯养成与品牌期待，可直接服务客户研究目标。  
- 主要风险：若无后续 IDI，深层动机与“信任/身份”类抽象理解可能不足以完全解释复杂认知差异（尤其益达的“自在”概念需要深访）。  
- 已做的控制：把情绪场景与功能场景并列、购物任务与实操吃用体验安排在后段以避免前期信息偏差，并鼓励多媒体材料降低文字负担。  
- 受访者负担可控性：日记模板尽量精简且可用语音/视频替代，购物与食用任务为指定日并提供替代回溯路径以容纳低频用户。  
- 数据可用性：设计能同时产出行为链（trigger→action→alternatives→choice logic→outcome）与对品牌的后设评价，便于后续形成可落地的communication/product方向。

## 6. 需要确认的问题（最多 3 条）
1. 是否会安排后续 IDI（或小样/深访）以承接 diary 中有代表性的受访者用于深挖“自在”情绪与身份认同？（若有，Diary 可少些深层追问；若无，Diary 需增加若干开放式引导题帮补深层动机。）  
2. 第5-6天的“吃用体验”是否要求受访者实际尝试并记录指定品牌（益达与绿箭）？还是仅记录自然使用/自带产品与回溯体验？（会影响是否需要提前寄样或补贴与合规说明。）  
3. 样本框层面：本次是否严格限定一线/新一线城市 Urban Strivers，还是允许覆盖部分二线城市？（影响场景代表性与渠道/可得性问题的观察重点。）

Agent 检核摘要（简短）：已完成对 Brief/Proposal 与 case_001 规则的校对与题目落地；主要待确认是否有 IDI 与是否需要指定品牌尝试以便最终细化任务与物料准备。
